# Information-Extractor
PEInfo – Extrator rápido de metadados e assinatura digital de executáveis Windows (PE32/PE32+)  • Nome, tamanho, SHA-1   • Data de compilação real   • Tamanho da imagem (SizeOfImage)   • Assinatura digital: Sim/Não + Assinante / Algoritmo (ex: Microsoft Corporation / sha256)  • Sem dependências externas   • Código-fonte limpo em C++ nativo (WinAPI)
